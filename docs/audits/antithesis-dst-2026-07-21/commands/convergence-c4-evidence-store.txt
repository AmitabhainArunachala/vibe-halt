## Shape

- vh_cli::receipts (crates/vh-cli/src/receipts.rs): PURE NDJSON codec —
  schemas vh-run-receipts-v1 (run.ndjson: manifest + per-universe +
  finding index) and vh-finding-bundle-v1 (self-contained replay bundle).
  No fs/clock/env; strict flat-record parser (rejects nesting, arrays,
  trailing garbage). 4 unit tests incl. hostile-detail escaping.
- crates/vh-cli/src/bundle.rs: boundary I/O — write_run_receipts +
  cmd_replay_bundle. Deny-list per-file exemption {std::fs} added with
  rationale in scripts/check_determinism_denylist.py (standing law §4:
  same-PR amendment, never a workaround); every other denied pattern
  still binds. Scan: PASS (9 crates, 35 files, 2 boundary skipped).
- vh run gains --out DIR (stdout + exit contract unchanged; --out
  conflicts with --universe, typed usage error). vh replay-bundle PATH:
  exit 0 REPRODUCED / 1 MISMATCH-or-DIVERGENT / 2 usage-or-malformed.
  Replay runs the universe TWICE and requires self-agreement before
  comparing to the recorded identity.
- Divergent universes are recorded in run.ndjson but never bundled — a
  bundle is a replay promise divergence cannot make.
- Tape digests (C1) not yet present: schema is additive; the field lands
  with C1 (§5 coupling noted in ledger).
- CLAUDE.md phantom retired: receipts written only where --out points,
  conventionally ~/.vibe-halt/, nothing implicit. Doc now matches a
  writer that exists.
